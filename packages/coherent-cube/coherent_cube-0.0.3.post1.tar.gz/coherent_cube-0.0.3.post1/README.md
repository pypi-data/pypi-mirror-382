# coherent-cube
Python controller for a Coherent Cube laser.

## Installation

```console
pip install coherent-cube
```

## Logging

Basic logging is provided via the [`logging` module](https://docs.python.org/3/library/logging.html).

## Examples

See the [`examples`](examples) folder.
