This folder contains STAC schema to validate EOProduct top-level attributes ("stac_discovery" attributes)

These metadata are validated following the STAC Item Specification from the "stac-spec" Github repository:

* Links to the STAC item specifications: https://github.com/radiantearth/stac-spec/blob/master/item-spec/item-spec.md#item-fields
  Corresponding jsonschema: https://github.com/radiantearth/stac-spec/blob/master/item-spec/json-schema/item.json

* Current release of the stac-spec repository: V1.1.0

* Link to the STAC extensions:
