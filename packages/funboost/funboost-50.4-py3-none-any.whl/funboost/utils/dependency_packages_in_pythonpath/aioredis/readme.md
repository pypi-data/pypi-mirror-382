## 2 aioredis.exceptions.py的TimeoutError在python3.11有问题

### 2.1 现在错误的

class TimeoutError(asyncio.TimeoutError, builtins.TimeoutError, RedisError):
    pass

class TimeoutError(asyncio.TimeoutError, RedisError):
    pass
