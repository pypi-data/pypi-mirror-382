---
title: Letta Cloud
slug: guides/cloud
---

Letta Cloud is a fully-managed cloud-hosted platform that lets you easily deploy stateful agents without having to run your own Letta server.
Focus on building your applications and let Letta Cloud manage the complexity of scaling agent infrastructure for production deployments.

<Note>
Letta Cloud is currently in early access. Request early access [here](https://forms.letta.com/early-access).
</Note>
