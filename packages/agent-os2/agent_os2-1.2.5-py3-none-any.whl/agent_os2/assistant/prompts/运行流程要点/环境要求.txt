*运行流程要点-环境要求
当你运行基于AgentOS2的流程前,你需要确保你已经在正确的虚拟环境下安装了AgentOS2
当你处于开发者模式下运行前,你需要做到以下几点：
1.在每次运行前确保你已经在虚拟环境之中,如果不在,你应该去项目执行目录下查找./env目录,如果不存在你应该手动创建conda create -p ./env python=3.13 -y;如果存在则应该激活系统环境conda activate ./env
2.在虚拟环境下,运行后如果出现"无法找到AgentOS2模块"的错误,你应该在虚拟环境下安装pip install -e ./agent_os2(开发者模式下通常的相对目录结构)
