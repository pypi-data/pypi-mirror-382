*Flow心智模型-资源控制[flow,内置模型处理器,构建时上下文]
Flow在调用完一个自己所属的子Agent后,会将该Agent的usage追加合并到自己的usage之中
可以在settings中设置max_tokens_limit,max_prompt_tokens_limit,max_completion_tokens_limit等参数来限制总的资源消耗
usage限制映射规则:
total_xxx -> max_xxx_limit
yyy -> max_yyy_limit
example:
Flow("test_flow",None,agents_key=...,expected_shared_context_keys=...,max_tokens_limit=131070)
