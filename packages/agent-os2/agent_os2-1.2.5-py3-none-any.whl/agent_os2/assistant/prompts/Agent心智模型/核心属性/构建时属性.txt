*Agent心智模型-核心属性-构建时属性[构建时上下文,setup方法,内置模型配置]
**会从构建时上下文settings中获取默认值**
is_debug:bool
debug_cache:dict[int,str]
debug_context:bool->是否启用上下文调试模式(bool)
usage:dict[str,Any]
stdout:Callable[[str,int|None,Any,str|None],None]->标准输出函数,默认为print函数
stdin:Callable[[str],Any]->标准输入函数,默认为input函数
model_config:ModelConfig->模型配置(ModelConfig),默认为gemini-2.5-flash大语言模型
这类属性可以在setup方法中覆盖
