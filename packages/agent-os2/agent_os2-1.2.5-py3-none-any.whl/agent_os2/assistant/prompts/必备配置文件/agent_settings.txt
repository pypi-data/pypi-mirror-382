*必备配置文件-agent_settings
存放位置:执行目录/aos_config/agent_settings.json或<agent_os2_path>/agent_settings.json(默认为内置examples的agents_key)
agent_settings.json中编排了所有agents_key下被注册的Agents和Flows
example(内置agent_settings.json):
{
    "story_generation_example":[
        "./examples/story_generate_example/agents",
        "./examples/story_generate_example/flows"
    ],
    "continue_story_example":[
        "./examples/story_generate_example/agents/continue_story"
    ],
    "assistant":[
        "./assistant/flows"
    ],
    "assistant/design_confirm":[
        "./assistant/agents/design_confirm"
    ]
}
